Практический кейс: ТЗ Копирайтеру на SEO-статью


Проект: [webstudia.online](https://webstudia.online)  
Тип страницы: Коммерческая посадочная страница услуги (Landing Page)  
Тема: Разработка Landing Page под ключ  

---


1. Базовая информация

- Основной ключевой запрос (Primary Keyword): заказать лендинг пейдж (Vol: 1200, KD: 25)
- Второстепенные ключи (Secondary Keywords): разработка лендинга цена, создание одностраничного сайта, сделать лендинг под ключ.
- Search Intent: Транзакционный (пользователь ищет исполнителя, чтобы заказать услугу и узнать цены).
- Объем текста: 3500 - 4500 символов без пробелов.
- Целевая аудитория (ЦА): Владельцы малого и среднего бизнеса, маркетологи, которым нужен быстрый запуск продукта или услуги. Боли: дорого, долго, не конвертит.


2. Мета-теги (заполняет SEO-специалист, не копирайтер)

- Title: Разработка Landing Page (Лендингов) под ключ | Цена от $500
- Description: Нужна высокая конверсия? Закажите разработку продающего Landing Page под ключ. Анализ ниши, современный дизайн и интеграция CRM. Оставьте заявку!
- H1: Разработка продающих Landing Page (Лендингов) под ключ для вашего бизнеса


3. Структура статьи (Обязательные заголовки)


1. H2: Почему вашему бизнесу нужен профессиональный Landing Page?
   - Задача блока: Описать боли клиента (старый сайт не продает, реклама сливается). Упомянуть ключ создание одностраничного сайта.
2. H2: Этапы разработки лендинга в нашей веб-студии
   - Задача блока: Показать экспертность (E-E-A-T). Описать шаги: Анализ конкурентов -> Прототип -> Дизайн -> Верстка -> Настройка аналитики/CRM.
3. H2: Сколько стоит сделать лендинг? (Цены и тарифы)
   - Задача блока: Ответить на интент разработка лендинга цена. Сделать таблицу или список с пакетами (Базовый, Бизнес, Премиум).
4. H2: Наши кейсы (До/После)
   - Задача блока: Доверие. Описать, как мы увеличили конверсию на реальном примере (текст-заглушка под картинки портфолио).
5. H2: Частые вопросы (FAQ)
   - H3: Сколько времени занимает создание лендинга?
   - H3: На какой платформе (CMS) вы работаете?
   - H3: Входит ли SEO-оптимизация в стоимость?


4. LSI-слова (обязательно использовать в тексте равномерно)

конверсия, лидогенерация, веб-дизайн, мобильная адаптация, A/B тестирование, Core Web Vitals, система управления (CMS), Call to action (CTA).


5. Требования к качеству (E-E-A-T)

- Никакой "воды" (штампов вроде "динамично развивающаяся компания").
- Текст должен быть разбит на короткие абзацы (3-4 строки).
- Обязательно использовать маркированные списки (минимум 2 штуки).
- Тональность (Tone of Voice): Экспертная, дружелюбная, с ориентацией на бизнес-показатели клиента (B2B).
